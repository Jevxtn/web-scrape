# Amazon Web Scraper

## Project Overview

This project is a web scraper built with Pythonüêç and BeautifulSoup to extract product information from Amazon. The scraper collects data such as product titles, prices, ratings, and availability, providing a useful tool for market analysis and price comparison.

## Features

- Extract product details like title, price, rating, and availability
- Handle pagination to scrape multiple pages of products
- Output data in a structured format (CSV or JSON)
- Easy to configure for different product categories or search queries

## Getting Started

### Prerequisites

To run this project, you'll need:

- Python 3.x installed on your machine
- Required Python packages: `requests`, `beautifulsoup4`, and `pandas` (for data handling)
